# 🎙️ UCL-Trade: Assistente de Voz para Análise Financeira

Bem-vindo ao repositório do **UCL-Trade**, um assistente virtual inteligente.

## 🎯 Objetivo
Transformar a interação com dados de mercado em uma experiência conversacional de voz, utilizando o estado da arte em IA.

## 🛠️ Stack Tecnológica
- **Linguagem:** Python
- **STT (Speech-to-Text):** OpenAI Whisper (Transcrição robusta)
- **LLM:** OpenAI GPT-4 (Cérebro do assistente)
- **TTS (Text-to-Speech):** Google gTTS (Sintetização da resposta)

## 📁 Organização
- `data/`: Base de conhecimento sobre trading e finanças.
- `docs/roteiro/`: Minha explicação detalhada sobre cada etapa do desenvolvimento.
- `src/`: Lógica de integração das APIs.

## 🚀 Como funciona
O assistente "lê" o arquivo `data/conhecimento_mercado.json` para dar respostas precisas sobre o mini-índice e estratégias de trading, mantendo a conversa fluida via voz.
